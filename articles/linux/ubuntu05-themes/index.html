Ubuntu Desktop 22.04 桌面主题配置

使用这么久 Ubuntu Desktop，本着不折腾的原则，简单介绍下自己的桌面主题配置。

安装 gnome-tweaks & chrome-gnome-shell

gnome 插件安装

Ubuntu 20.04 默认使用的是 gnome 的桌面环境，gnome 插件的下载地址：
gnome extensions
直接搜索需要的插件安装即可。

命令行管理 gnome shell extensions

gnome 主题安装

gnome 主题下载地址：
gnmoe-look
直接选择合适的主题，点击 install 即可，主题会自动安装到 ~/.themes/ 目录下。或者点击 Download 下载解压后自行放到对应文件夹下。
然后打开 tweaks -> 外观 -> 选择合适的主题即可。

图标：和上面类似，下载后在 tweaks 里设置图标即可。目录 ~/.local/share/icons/
